Detailed Description

Class that contains a deployable instance of the Evaluator Platform.

The Evaluator class behaves single-handedly as the Evaluator platform itself. It contains multiple interfaces such as Courses, Problem_repo and Users. Evaluator itself can be deployed elsewhere, and only one evaluator instance can be on use in a single processor thread.
